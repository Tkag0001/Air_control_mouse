# Air_control_mouse
Use hand to control mouse PC.
## Clone repository
```
git clone https://github.com/Tkag0001/Air_control_mouse.git
```
## Setup enviroment
Create enviroment:
```
cd Air_control_mouse
python -m ven ./my_venv
```

Active enviroment:
```
cd my_venv
Scripts\activate
```

## Install requirment packages
```
pip install -r requirements.txt
````

## Use app
```
python app.py
```
